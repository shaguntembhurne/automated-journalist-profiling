# Automated Journalist Profiling

This repository contains sample outputs from the automated journalist profiling system proposed in our research. The system collects publicly available web data, processes unstructured information, and generates structured journalist profiles using an agentic pipeline with large language models.

The results included here demonstrate the effectiveness of the pipeline across multiple journalists with varying levels of online presence.

All outputs are provided for verification and reproducibility purposes.

For detailed methodology and evaluation, refer to the associated research paper.
